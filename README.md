# Groq API Demo

A simple Node.js project demonstrating how to connect to the Groq API using Node's built-in environment file parsing.

## Getting Started

1. Copy the `.env.example` file to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Populate the `GROQ_API_KEY` in `.env` with your API key.
3. Run the script:
   ```bash
   node --env-file=.env index.js
   ```
